## 1. 世界观

### 1.1 世界构成

| 世界 | 说明 | 游戏定位（区域段） |
|------|------|-------------------|
| **现世** | 现代人类社会（空座町一带），主角原本所在 | 序章·觉醒（区域 1~3 浅打段） |
| **尸魂界** | 死神与流魂街所在，灵子浓度高 | 始解/卍解段（区域 4~9） |
| **虚圈** | 虚的领地，荒芜沙漠与虚夜宫 | 虚化段（区域 10~12） |
| **完现术据点** | 完现术者组织 XCUTION 的据点 | 完现术段（区域 13~15） |
| **无形帝国** | 灭却师·友哈巴赫的圣文字军团 | 圣文字段（区域 16~18） |
| **地狱** | 咎人沉沦之地，狱解之力源头 | 狱解段（区域 19~21） |

### 1.2 核心设定（Bleach 向）

- **斩魄刀**：死神力量的具现。觉醒即职业选择，每把刀有独立成长树（始解/卍解）。
- **鬼道**：操纵灵子的术法体系（破道/缚道/辅助），作为副职业三系自由搭配。
- **虚（Hollow）**：迷失灵魂堕落而成的怪物，是主要敌人。被灵子吸引、吞噬灵魂。
- **元素相克**：火克风、风克土、土克水、水克火（四系循环）。
- **主角身份**：空座町的代理死神——街头斗殴时捡到一把斩魄刀、触碰即共鸣的少年。从零觉醒、一路净化虚、对抗虚圈与无形帝国，最终抵达地狱深渊完成狱解。

## 2. 历史与势力背景

- **尸魂界**：静灵庭十三番队维持秩序，流魂街是普通灵魂居所。
- **虚圈**：蓝染（原静灵庭队长）曾盘踞虚夜宫、培育大虚；现由虚群统治。
- **无形帝国**：友哈巴赫率圣文字军团，发动圣别吞噬死神力量。
- **地狱**：咎人（罪魂）受罚之地，狱解之力源自此处。

> 注：具体人物与事件以代码 `Zones.ts` 的 NPC `dialogue` 与 `QuestData.ts` 主线为准，本文档只定义世界框架。

## 4. 核心冲突

### 主线：从现世觉醒，一路净化虚、深入各势力终局

主角在现世捡到斩魄刀后，被卷入灵子涌动的世界。他/她从零觉醒斩魄刀、修习鬼道，肩负起净化虚、逐步推进到尸魂界、虚圈、完现术据点、无形帝国，最终抵达地狱深渊完成狱解的使命。

**主线推进结构（21 区域线性）**：
1. 现世觉醒，触碰斩魄刀之门（始解觉醒）
2. 尸魂界·流魂街，完成卍解觉醒
3. 虚圈，完成虚化觉醒
4. 完现术据点，完成完现术觉醒
5. 无形帝国，完成圣文字觉醒
6. 地狱深渊，完成狱解觉醒（终局）
7. 揭开自身斩魄刀的终极秘密

### 暗线

- 自身斩魄刀的前世（每把刀的专属故事）
- 虚圈与无形帝国的真正目的
- 地狱为何向现世伸出触手

## 8. 设计决策

- [x] 世界观重构：旧「灵渊」设定废弃，改为 Bleach（死神）题材（v0.9，2026-07-05）
- [x] 区域数量：首发 **21 个区域**（v0.9，原 7 个），详见 09-地图探索系统
- [x] 区域推进：**剧情解锁**（非「区域解放」机制），边界检测切换
- [x] 组队 / 同伴系统：**首发单主角**（剧情 NPC 可临时协助，不入队）
- [x] 结局方向：主线终局分支（详见 08-任务系统 §9）
- [x] 力量体系：浅打 → 始解 → 卍解 → 虚化 → 完现术 → 圣文字 → 狱解，共 7 段，由主线任务 `addUnlock` 激活
