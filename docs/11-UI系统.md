## 一、设计总则

### 1.1 视觉风格

| 要素 | 规格 |
|------|------|
| **风格** | 2D卡通，暖色基调，手绘感边框，毛笔字风格标题 |
| **分辨率** | 适配基准 1920×1080，支持 1280×720~2560×1440 缩放 |
| **配色** | 深棕底色 + 米白文字 + 金色高亮 + 红色强调（和风/东方幻想） |
| **字体** | 中文：站酷文艺体 / 日系毛笔风格；数字：等宽像素风 |
| **文字渲染** | 所有Text必须加 `padding: { y: 2 }`（大字号24px+加y:3~4），长文本加 `wordWrap: { width: N }`，禁止用 `setFixedSize` 限制高度（会导致中文垂直截断） |
| **动画** | 面板滑入（0.2s ease-out），按钮hover放大1.05x，切换淡入淡出（0.15s） |
| **音效** | 按钮hover轻响，确认点击清脆木鱼声，面板开关纸张翻页声 |

| 状态 | HP条表现 | 画面效果 |

| HP>50% | 绿色 | 正常 |

### 11.1 中文文字渲染规范（重要）

Phaser 3 的 Text 对象渲染中文时，默认没有内边距，中文字符上下边缘容易被裁切。以下是经过多次调试总结的规范：

| 规则 | 说明 | 示例 |
|------|------|------|
| **必须加padding** | 所有Text对象的style中必须包含 `padding: { y: 2 }` 以上 | `{ fontSize: '14px', color: '#ccc', padding: { y: 2 } }` |
| **大字号加大padding** | 24px以上字号需要 `padding: { y: 3 }` ~ `{ y: 4 }` | `{ fontSize: '96px', padding: { y: 4 } }` |
| **禁止setFixedSize限制高度** | `setFixedSize(w, h)` 会裁切超出高度的中文，如需限制宽度用 `wordWrap` | ❌ `text.setFixedSize(440, 16)` → ✅ `wordWrap: { width: 440 }` |
| **长文本加wordWrap** | 描述性文字、技能说明等需加 `wordWrap: { width: N }` 防止水平溢出 | `{ wordWrap: { width: 280 } }` |
| **setMaxWidth不存在** | Phaser 3 Text 没有 `setMaxWidth` 方法，用 `wordWrap` 代替 | ❌ `text.setMaxWidth(280)` → ✅ `wordWrap: { width: 280 }` |
