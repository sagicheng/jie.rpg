```

                        │       玩家角色           │
                        └───────────┬─────────────┘
                                    │
            ┌───────────────────────┼───────────────────────┐
            │                       │                       │
     ┌──────▼──────┐         ┌──────▼──────┐         ┌──────▼──────┐
```

**核心逻辑**：斩魄刀决定"你是谁"（职业），鬼道决定"你还会什么"（副职业）。两者独立成长，自由搭配。

## 二、元素系别

### 2.1 四系总览

斩魄刀分属四大系别：**风、火、水、土**。系别决定斩魄刀的内核特性与战斗风格。

| 系别 | 灵魂关键词 | 战斗风格 | 代表特征 |
|------|-----------|----------|----------|
| **火** | 烈、焚、爆 | 高爆发、灼烧持续、范围清场 | 一刀定江山 |
| **水** | 静、凝、透 | 控制冻结、持续消耗、以柔克刚 | 敌不动，我先动 |
| **风** | 疾、散、锐 | 高速多段、闪避反击、逐个击破 | 天下武功，唯快不破 |
| **土** | 固、沉、镇 | 高防御、眩晕压制、以守为攻 | 不动如山 |

### 2.2 元素相克

```
火 ──克──→ 风 ──克──→ 土 ──克──→ 水 ──克──→ 火
```

| 克制关系 | 效果 |
|----------|------|
| 火克风 | 火属性对风属性伤害×1.5 |
| 风克土 | 风属性对土属性伤害×1.5 |
| 土克水 | 土属性对水属性伤害×1.5 |
| 水克火 | 水属性对火属性伤害×1.5 |

> 克制倍率 1.5x，抵抗 0.5x，同属性 0.75x。

## 三、斩魄刀系统（主职业）

### 3.1 觉醒机制

- 玩家在序章结束后进入斩魄刀觉醒剧情
- 觉醒结果 = 职业选择。一旦觉醒，不可更换
- 从自身灵魂中唤醒一把斩魄刀——每把刀都承载着专属的成长故事与前任死神的意志
- 每把刀附带前任主人的简短故事

### 3.2 成长阶段

| 阶段 | 解锁条件 | 效果 |
|------|----------|------|
| **浅打** | 初始获得 | 基础形态，1个基础技能 |
| **始解** | 觉醒斩魄刀时解锁（序章末剧情节点，无等级/区域门槛；`GameState.addUnlock('shikai')`） | 刀解放，2个始解技能 + 始解被动 |
| **卍解** | 完成卍解主线任务解锁（`GameState.addUnlock('bankai')`） | 最终解放，1个卍解奥义 + 卍解被动，战斗中限时爆发（持续3回合，每场1次） |
| **虚化** | 完成虚化主线任务（`addUnlock('hollow')`） | 虚化形态，额外爆发技能 |
| **完现术** | 完成完现术主线任务（`addUnlock('fullbring')`） | 完现术装备技能 |
| **圣文字** | 完成圣文字主线任务（`addUnlock('schrift')`） | 圣文字能力 |
| **狱解** | 完成狱解主线任务（`addUnlock('hell')`） | 狱解形态，终局能力 |

### 3.7 异常状态系统（新增v0.7）

**属性**：
- **异常命中率(statusAcc)**：由斩魄刀决定，不可加点，在属性面板显示
- **异常抗性(statusRes)**：敌人属性（杂妖0%/恶妖5%/妖将15%/妖王30%）

**命中公式**：`最终概率 = 基础概率 + 斩魄刀加成 - 敌人抗性`，上限95%/下限5%

**适用范围**：鬼道控制技能（缚道全系）和斩魄刀概率异常技能共用同一套公式。

## 四、鬼道系统（副职业）v0.3.0 已实现 — v0.7 重做为天赋树v3.0

> **当前状态（v0.7）**：参照LOL早期天赋树重做。三系各4列×5层=20节点，共60节点。纯层锁（T1=0, T2=5, T3=10, T4=15, T5=20点该系），无节点间前置。三系完全自由学习，无主修限制。被动属性（MATK+%/DEF+%等）自动集成到角色面板。MP消耗每级递增12%。按 **K键** 打开天赋面板。战斗中直接使用所有已学主动技能。

### 4.1 设计理念

鬼道独立于斩魄刀职业。三系完全自由学习，无主修限制。

**核心规则**：
- 三系均可投入点数，不限制
- 每升1级获得1点鬼道点
- 按层解锁：T1=0, T2=5, T3=10, T4=15, T5=20点该系即可
- 同一回合中：斩魄刀技能 vs 鬼道技能，只能择一使用
- 鬼道均为**直接释放**，无需蓄力回合
- 战斗中释放鬼道时播放**咏唱文字特效**（纯视觉表现，不影响战斗节奏）
