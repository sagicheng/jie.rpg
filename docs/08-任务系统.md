## 一、系统定位

任务系统是《解》的**内容驱动引擎**（P1），承载主线剧情推进、支线世界构建、以及日常玩法循环。

**核心规则**：
- 主线 = 线性推进，不可跳过，推动区域推进（剧情解锁）
- 支线 = 可选，补充世界观和养成资源
- 日常 = 重复可刷，产金币/材料/经验
- 所有任务无时间限制（除特殊限时任务外）

## 三、主线任务

### 3.1 章节结构

> 设计说明：以下 7 章 / 35 节点沿用 pre-Bleach 草稿的章节与区域命名（序章误入灵渊、萌芽村、腐木之森等）。实际实现时将按 Bleach 21 区域线性世界重新映射（序章≈区域1~3 浅打段，第一章≈区域4~6 始解段……详见 09-地图探索系统 §二），节点数量与剧情职能保留。以代码 `QuestData.ts` 为最终真值源。

7 个力量阶段 = **21 个区域（线性推进）**。章节映射以代码 `QuestData.ts` 为准，下表为区域→力量体系的剧情分段（区域名取自 `Zones.ts`）：

| 段位 | 区域 | 区域名（代码） | 剧情定位 |
|------|------|--------------|----------|
| 浅打（1~3） | 1~3 | 浦原商店街 / 空座高校 / 河川敷 | 现世觉醒，捡刀→始解之门 |
| 始解（4~6） | 4~6 | 润林安 / 戌吊 / 草鹿 | 尸魂界·流魂街，始解觉醒 |
| 卍解（7~9） | 7~9 | 一番队舍 / 技术開発局 / 真央灵术院 | 静灵庭，卍解觉醒 |
| 虚化（10~12） | 10~12 | 白砂原 / 黑腔深部 / 虚夜宫 | 虚圈，虚化觉醒 |
| 完现术（13~15） | 13~15 | 战迹 / XCUTION基地 / 完现术总本山 | 完现术据点，完现术觉醒 |
| 圣文字（16~18） | 16~18 | 影之领域 / 星十字宫 / 银架城 | 无形帝国，圣文字觉醒 |
| 狱解（19~21） | 19~21 | 咎人之门 / 无间 / 终焉之渊 | 地狱，狱解觉醒（终局） |

> 区域推进方式：**剧情解锁**（非旧设定的「区域解放」机制）。每完成一段主线节点，`GameState.addUnlock(...)` 激活对应力量体系，并解锁下一区域入口（边界检测切换，详见 09-地图探索系统 §四）。具体节点数量与职能以代码 `QuestData.ts` 为唯一真值源。

## 四、支线任务

### 4.1 支线分类

| 类型 | 说明 | 触发NPC | 数量 |
|------|------|---------|------|
| **角色支线** | 据点NPC的个人故事 | 老铁匠、守卫队长、鬼道师等 | 6~8 |
| **探索支线** | 在地图中隐藏触发 | 特定地点、物品互动 | 5~7 |
| **斩魄刀故人支线** | 与斩魄刀故人成员遗物相关的任务 | 斩魄刀具现体提示 | 5 |
| **妖魔支线** | 猎杀特定妖魔/收集图鉴 | 任务板 | 5~7 |
| **制造支线** | 制造特定物品交付 | 工匠NPC | 4~6 |

### 4.2 支线示例

#### 角色支线示例

| 任务 | NPC | 区域 | 简述 |
|------|-----|------|------|
| 老铁匠的遗憾 | 老铁匠 | ①→⑥ | 老铁匠曾为斩魄刀故人打造装备，但有一件未完成。收集材料帮他完成→获得传说道具 |
| 守卫的觉醒 | 守卫队长 | ①→⑤ | 守卫队长一直想觉醒斩魄刀。在区域推进过程中，他逐渐找到自己的路 |
| 鬼道师的过去 | 鬼道师 | ①→④ | 鬼道师曾是斩魄刀故人的鬼道支援者。推进润林安后，她讲述斩魄刀故人在流魂街的经历 |

#### 斩魄刀故人支线示例

| 任务 | 触发条件 | 简述 | 奖励 |
|------|---------|------|------|
| 炎柱的残火 | 装备流刃若火+进入一番队舍 | 在特定地点触发炎柱残留的灵子记忆 | 流刃若火专属技能 |
| 冰帝的遗迹 | 发现冰帝遗留物品 | 追踪冰帝在终焉之渊入口的痕迹 | 冰轮丸专属被动 |
| 队长的选择 | 第七章·节点4 | 了解队长最后的决定 | 全斩魄刀通用技能 |

### 4.3 支线设计原则

- 每个支线不少于3个步骤（避免"跑腿一句话"的廉价感）
- 奖励与区域进度匹配（不在前期给后期装备）
- 角色支线与主线进程绑定（区域不解锁无法推进支线）
- 支线之间不相互依赖（避免卡任务链）

## 五、日常任务

### 5.1 日常刷新规则

- 每日0:00（服务器时间）刷新
- 同时最多接取3个日常任务
- 每日完成任务上限：5个
- 未完成的日常不累积到次日

### 5.2 日常任务池

| 类型 | 任务示例 | 奖励 |
|------|---------|------|
| 讨伐 | 击败X只指定妖魔 | 金币+经验 |
| 采集 | 收集X个指定材料 | 材料包+金币 |
| 制造 | 制造X件指定装备/药品 | 制造经验+强化石 |
| 探索 | 在指定区域找到X个目标点 | 金币+传送符 |
| 救援 | 护送NPC穿越妖魔区 | 经验+稀有材料 |

### 5.3 日常难度匹配

日常任务的难度和奖励随玩家所在区域动态调整：

| 玩家当前区域 | 日常任务区域 | 奖励倍率 |
|------------|------------|---------|
| ①~② | ①~② | ×1.0 |
| ③~④ | ②~④ | ×1.5 |
| ⑤~⑥ | ④~⑥ | ×2.0 |
| ⑦ | ⑤~⑦ | ×3.0 |

## 八、任务链完整性

### 8.1 主线-支线联动

部分支线在主线特定节点后才解锁，形成内容密度高峰：

| 主线节点 | 解锁支线 | 目的 |
|---------|---------|------|
| 第一章完成 | 老铁匠的遗憾、守卫的觉醒 | 前期养成资源补充 |
| 第二章完成 | 鬼道师的过去、2个探索支线 | 鬼道系统深化 |
| 第三章完成 | 斩魄刀故人支线×2 | 世界观扩展 |
| 第五章完成 | 斩魄刀故人支线×2、角色支线高潮 | 终章前的情绪铺垫 |
| 第六章完成 | 斩魄刀故人支线最终章 | 斩魄刀故人全员故事收束 |

### 8.2 内容密度控制

```
序章 ──── 第一章 ──── 第二章 ──── 第三章
 ████      ████       ██████     ████████
(4节点)   (4节点)     (4节点)     (4节点)
 +0支线    +2支线      +3支线      +4支线

第四章 ──── 第五章 ──── 第六章 ──── 第七章
████████   █████████   ██████████  ██████████
```

> 支线在中期（3~5章）密度最高，终章收束——玩家注意力回归主线。

## 十、待决策事项

- [x] 组队/同伴系统：**首发不设同伴**（与02/10决策一致）。支线全部围绕单主角展开。
- [x] 不可逆支线：**不设**。所有支线可随时返回完成，避免玩家焦虑。想增加抉择感→通过隐藏任务和选择分支实现（§9.1/§9.2）。
- [x] 周常任务：**增设**。每周刷新3个周常（精英讨伐/区域探索/制造挑战），奖励=日常×5倍。周常可在一周内任意时间完成。
- [x] 隐藏任务提示：**模糊暗示**。在NPC对话或斩魄刀故人笔记中埋藏线索，不做UI标记。完全隐藏会让99%玩家错过。
- [x] 结局分支与多周目：**不支持New Game+**（首发）。两个结局均可通过"返回标题→继续"在最终选择前重新体验。后续版本可评估。
- [x] 任务对话语音：**纯文本**。2D卡通RPG文本阅读为主流，语音成本高且非必需。
