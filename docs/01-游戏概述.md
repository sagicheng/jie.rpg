## 2. 核心概念

主角是一名成绩不好的普通高中生。一次街头斗殴中，随手捡起一把斩魄刀——却因此觉醒了死神之力。
在这个虚横行、灵子涌动的世界里，斩魄刀与鬼道是人类（代理死神）对抗虚的唯一手段。

主角从零开始觉醒斩魄刀、修习鬼道，肩负起净化虚、一路深入尸魂界/虚圈/无形帝国/地狱的使命。

**一句话描述**：捡起那把刀的瞬间，他的人生再不属于自己。

---

## 3. 参考与差异化

### 参考《飘流幻境》的核心要素
- 2D俯视角地图探索
- 经典回合制战斗
- 丰富的制造/合成系统
- 多元素属性克制
- 休闲与冒险并存

### 本作核心差异

**斩魄刀 = 职业选择**
觉醒不同的斩魄刀等同于选择不同的职业路径。全部斩魄刀归属四大系别——**风、火、水、土**。每系含6把斩魄刀，共24把（含原作经典+原创设计）。斩魄刀决定战斗风格、属性倾向和专属技能树。始解和卍解是职业进阶。

**鬼道 = 副职业**
破道（攻击）、缚道（控制）、回道（治疗）作为副职业系统，独立于斩魄刀职业。玩家可自由搭配——比如流刃若火（火系主职业）+ 缚道专精（控制副职）。

**唯一武器系统**
斩魄刀不可更换，但每把都有一条深度进化树：浅打 → 始解 → 卍解。成长过程比"换装备"更有仪式感。

## 5. 核心玩法循环

```
探索异世界地图（自由行走+与NPC交互）
  → 遭遇敌人（暗雷/明雷）
    → 回合制战斗（指令选择）
      → 获得经验/灵子/材料
        → 角色成长 + 斩魄刀进化 + 鬼道收集
          → 探索更广阔/高难区域
            → 剧情推进
```

**核心驱动**：
- **斩魄刀养成**——从浅打到始解，再到卍解，全程陪伴的成长线
- **鬼道收集与构筑**——自由搭配技能组合
- **世界探索**——揭开异世界地图与故事
- **制造合成**——采集材料、打造道具装备

## 8. 差异化设计点

1. **斩魄刀=职业**——觉醒即是选择，每把刀一条专属成长树，拒绝同质化
2. **鬼道副职业**——三系自由搭配，不绑主职业，组合策略深度
3. **元素相克**——火克风、风克土、土克水、水克火，四系循环克制
4. **区域推进**——完成主线节点后剧情解锁下一区域（边界检测切换），逐段获得新功能/新NPC（详见09-地图探索系统）
