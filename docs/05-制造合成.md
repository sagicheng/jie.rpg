## 一、系统定位

制造合成为《解》的核心养成系统之一（P0）。在斩魄刀不可更换的前提下，制造合成是玩家提升战力的主要途径。

**核心规则**：
- 斩魄刀 = 不可更换（灵魂绑定）
- 防具 + 饰品 = 制造获取（数值堆砌）
- 药品 + 道具 = 制造获取（消耗品补给）
- **制造技能统一**，无等级/无熟练度，有配方有材料即可制造
- **制造产物可交易**（自由流通）
- **背包容量不限制**

## 六、制造流程

```
获取配方 → 收集材料 → 前往据点制造台
                          │
                          ▼
                    选择配方
                          │
                          ▼
                    确认材料（不足则提示缺失）
                          │
                          ▼
                    开始制造
                          │
                    ┌─────┴─────┐
                    ▼           ▼
                 成功         失败
                  │           │
             产出物品     材料部分损失
             品质随机     保留配方可用
```

**失败规则**：
- 基础成功率由配方决定（通常70%~95%）
- 失败后消耗50%材料，不产出物品
- 配方永久保留，可重复尝试

## 七、装备耐久

### 7.1 耐久机制

所有防具和饰品均有耐久度：

| 品质 | 最大耐久 | 说明 |
|------|---------|------|
| 白 | 100 | 每场战斗-1 |
| 绿 | 130 | 每场战斗-1 |
| 蓝 | 160 | 每场战斗-1 |
| 紫 | 200 | 每场战斗-1 |
| 金 | 250 | 每场战斗-1 |

- 耐久归零 → 装备属性失效，但仍可修理恢复
- 被击倒（HP归零）→ 全身装备额外-5耐久
- Boss战失败 → 全身装备额外-10耐久

### 7.2 修理系统

在据点制造台进行修理：

| 品质 | 修理材料 | 金币花费 |
|------|---------|---------|
| 白 | 矿石×1 | 50 |
| 绿 | 矿石×2 | 150 |
| 蓝 | 矿石×3 + 灵质×1 | 500 |
| 紫 | 矿石×5 + 灵质×2 | 1500 |
| 金 | 灵晶矿×2 + 灵质×3 | 5000 |

> 消耗材料修理 → 增加制造系统的持续需求 → 材料始终有价值。

## 十一、分解系统 ✅ 已实现

### 11.1 系统定位

分解是将不需要的装备转化为材料的手段——让"垃圾装备"有价值，形成**制造→装备→分解→材料→制造**的循环闭环。

### 11.2 分解规则

| 规则 | 说明 |
|------|------|
| **分解对象** | 所有可装备的防具、饰品（斩魄刀不可分解） |
| **分解地点** | 据点制造台 |
| **返还机制** | 按装备品质返还材料，数量为该装备制造配方的30%~60% |
| **不可撤销** | 分解后装备消失，不可恢复 |

### 11.3 分解返还表

| 品质 | 返还比例 | 返还材料 |
|------|---------|---------|
| 白 | 30% | 制造该装备所需的基础材料×30%（向下取整，至少1个） |
| 绿 | 35% | 制造所需材料×35% + 1个该区域特殊材料 |
| 蓝 | 40% | 制造所需材料×40% + 1~2个该区域特殊材料 |
| 紫 | 50% | 制造所需材料×50% + 2个该区域稀有材料 |
| 金 | 60% | 制造所需材料×60% + 3个该区域稀有材料 |

> 例：分解一件紫装"润林之铠"（制造需矿石×5 + 灵质×2），返还矿石×3（5×50%向上取整）+ 灵质×1（2×50%）+ 润林安稀有材料×2。

### 11.4 分解与强化的交互

| 装备状态 | 分解规则 |
|---------|---------|
| 未强化 | 按品质返还（§11.3） |
| 已强化 | 额外返还强化石（=已消耗强化石×30%） |
| 已精炼 | 不返还精炼石（精炼石一次性消耗） |
| 耐久受损 | 耐久度不影响分解返还 |

> 强化保护符不返还。

### 11.5 设计意图

- 白色垃圾装 = 微量材料回收，聊胜于无
- 绿装 = 过渡期获取基础材料的手段
- 蓝/紫装 = 主要材料回收来源，分解替换下来的装备支撑下阶段制造
- 金装 = 高返还比例，但金装不可交易，分解为终极选择

## 十二、待决策事项

- [x] 修理方式：**仅据点制造台**。强化"回据点"的节奏感，也为据点内的各种交互创造机会。不设野外修理道具。
- [x] 强化石品质：**单一品质**。不区分普通/精良/稀有。简化系统，强化石本身就是通用货币型材料。
- [x] 精炼重铸石：获取途径——Boss掉落（20%概率）、日常任务奖励（每周上限2个）、终焉之渊宝箱。商店售价：5,000金币/个。
- [x] 装备丢弃/卖店：**可以卖店**。白色装备回收价=制造材料价的20%，绿色以上装备+分解更划算。卖店是"懒得分解"的快捷选项。
- [x] 分解返还比例：当前保守值（白30%~金60%），数值测试后根据材料经济再微调。当前作为设计基准足够。
