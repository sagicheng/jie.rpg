## 一、系统定位

角色系统是《解》的**底层成长框架**（P0）。所有战斗、养成、探索的数值根基。

**核心规则**：
- 单主角，无固定同伴（组队系统待定）
- 玩家命名，性别可选
- 等级×属性点分配 = 角色基础成长
- 斩魄刀决定属性成长倾向（03已定）
- 装备+强化+精炼叠加在基础属性之上（05/06已定）
- 灵压（MP）是战斗资源也是叙事元素

## 四、属性体系

### 4.1 基础属性

| 属性 | 简称 | 说明 | 初始值(Lv1) |
|------|------|------|-----------|
| 生命值 | HP | 归零则战斗失败 | 100 |
| 灵压 | MP | 技能/鬼道消耗资源，归零可继续普攻 | 50 |
| 攻击力 | ATK | 物理攻击，影响普攻和物理技能 | 10 |
| 防御力 | DEF | 减少物理伤害 | 8 |
| 魔法攻击 | MATK | 魔法攻击，影响魔法技能和鬼道强度 | 10 |
| 魔防 | MDEF | 减少魔法伤害 | 8 |
| 速度 | SPD | 行动顺序+部分技能加成 | 10 |
| 暴击率 | CRT | 基础5%，不随等级成长 | 5% |
| 暴击伤害 | CDM | 基础150%，不随等级成长 | 150% |

### 4.2 属性成长

**每升1级获得6个属性点**，玩家手动分配到 HP / MP / ATK / DEF / MATK / SDF / SPD。

| 属性 | 每点换算 | Lv1→Lv70满点投入效果 |
|------|---------|---------------------|
| HP | +15 HP | +6,210 HP（414点全投） |
| MP | +5 MP | +2,070 MP |
| ATK | +1 ATK(物理) | +414 ATK |
| DEF | +1 DEF | +414 DEF |
| MATK | +1 MATK(魔法) | +414 MATK |
| SDF | +1 SDF | +414 SDF |
| SPD | +1 SPD | +414 SPD |

> 实际分配通常不会全投单一属性。正常练级下各属性均衡或侧重分配。

### 4.3 斩魄刀成长倾向

投入的属性点经斩魄刀倾向倍率换算后才是最终基础属性值：

```
最终基础属性 = 等级自带基础值 + Σ(投入点数 × 斩魄刀倾向倍率)

例：流刃若火，Lv20，ATK投入30点
  ATK = 10 + 30×1.5 = 55
  DEF = 8 + 投入点×0.8
```

倾向倍率表（已在03文档完整列出），此处速查：

| 斩魄刀 | ATK | DEF | MATK | SDF | SPD |
|--------|-----|-----|-----|-----|-----|
| 流刃若火 | 1.5 | 0.8 | 1.2 | 0.8 | 0.9 |
| 斩月 | 1.5 | 0.8 | 0.7 | 0.8 | 1.3 |
| 岩铁碎 | 0.9 | 1.5 | 0.6 | 1.4 | 0.6 |
| 春时雨 | 0.7 | 1.1 | 1.5 | 1.2 | 0.8 |
| 千本樱 | 1.3 | 0.7 | 1.0 | 0.7 | 1.5 |

## 五、灵压系统

### 5.1 设计理念

灵压不只是蓝条。它是角色与灵子世界（尸魂界/现世）连接的度量。

| 层面 | 说明 |
|------|------|
| **战斗资源** | MP，技能/鬼道的消耗来源 |
| **探索资源** | 高灵压区域（虚圈/地狱）中持续衰减，需找灵子泉眼恢复 |
| **剧情符号** | 灵压的高低在某些剧情节点触发不同的对话选项 |

### 5.2 灵压恢复

| 方式 | 恢复量 | 条件 |
|------|--------|------|
| 战斗胜利 | 最大MP的15% | 每场 |
| 据点休息（旅店） | 满恢复 | 免费 |
| MP回复药 | 固定值 | 道具使用 |
| 灵子泉眼（野外） | 持续恢复 | 站立在泉眼旁 |
| 春时雨被动 | 每回合自动回复 | 被动技能 |

> 野外没有自动回复。需要用药或回据点——增加探索的资源管理深度。

## 六、角色状态

### 6.1 正常状态

| 状态 | 条件 | 说明 |
|------|------|------|
| 正常 | HP>0 | 可自由行动 |
| 战斗不能 | HP=0 | 见§6.2 |

### 6.2 战斗不能（死亡）

| 结果 | 说明 |
|------|------|
| 被击倒时 | 战斗失败。选择「重新挑战」或「返回据点」 |
| 重新挑战 | 在Boss房/遇敌点重新战斗，全队满状态恢复 |
| 返回据点 | 传送至最近据点，HP/MP全满 |
| 装备耐久惩罚 | 全身装备耐久-5（普通）/ -10（Boss战），05文档已定 |
| 经验惩罚 | **无**——不设死亡扣经验 |

> 不设经验惩罚的理由：装备耐久已经是惩罚。重复扣经验会鼓励"安全刷怪"而非"挑战Boss"——与游戏调性不符。

### 6.3 异常状态

战斗中的异常状态由04文档完整定义（灼烧/冻结/中毒/寄生/减速/眩晕/禁锢/嘲讽/恐惧/降攻/降防/降灵压），角色系统不做额外叠加。

## 八、洗点系统

### 8.1 规则

| 项目 | 说明 |
|------|------|
| 洗点条件 | 任意据点，消耗金币 |
| 洗点对象 | 已分配的所有属性点（不含等级基础值） |
| 费用 | 首次免费，后续 = 角色等级 × 200 金币 |
| 限制 | 无次数限制 |

### 8.2 设计意图

斩魄刀不可更换，但加点可以。允许玩家尝试不同build（全攻流刃若火 vs 均衡流刃若火），降低"加错点"的挫败感。

## 九、存档系统

### 9.1 存档方式

| 方式 | 说明 |
|------|------|
| **自动存档** | 进入新区域时、Boss战前、完成主线节点时 |
| **手动存档** | 在据点旅店"休息"时触发存档（3个存档位） |
| **快速存档** | 暂停菜单中（1个快速存档位，覆盖式） |

### 9.2 存档内容

- 角色等级、属性、装备
- 斩魄刀状态、鬼道技能
- 背包、金币
- 任务进度
- 区域推进状态
- 妖魔图鉴收集度
- 游戏时长
